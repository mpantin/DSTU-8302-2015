### Overview

This repository provides a **custom Citation Style Language (CSL) file** implementing the Ukrainian national bibliographic standard **DSTU 8302:2015**.

The style is intended for use with **Zotero** and supports preparation of **academic articles, theses, dissertations, conference papers, and technical reports**.

### Features

- Numeric in-text citations in square brackets: `[1]`
- Bibliography formatted according to **DSTU 8302:2015**
- Correct handling of:
  - Multiple authors using *‚Äú—Ç–∞ —ñ–Ω.‚Äù*
  - Journal articles, books, chapters, conference papers, reports, and online sources
  - DOI, URL, ISBN, access dates
  - Volume, issue, pages, editions, series
- Improved formatting for conference references:
  - separation of conference title, abbreviation, place, and year
- Fixes common CSL issues:
  - duplicate punctuation (`—Ç–∞ —ñ–Ω. .`)
  - missing spaces between bracketed elements (`)(`)
  - concatenation errors (e.g. `2019DOI:`)

### Files

- `dstu-8302-2015.csl` ‚Äî main CSL style file

### Installation

1. Download the CSL file from this repository.
2. Open **Zotero** ‚Üí **Settings ‚Üí Cite ‚Üí Styles**
3. Click **"+"** and select the CSL file
4. Choose **DSTU 8302:2015** as your citation style

### Intended Audience

- Researchers and PhD students
- University staff and librarians
- Anyone preparing academic texts according to Ukrainian standards

### Disclaimer

This CSL style is provided **as-is**.
Always verify the final bibliography against the official **DSTU 8302:2015** standard and your institution‚Äôs requirements.
